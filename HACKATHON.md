# Polkadot Solidity Hackathon (APAC, 2026)

## Introduction
The **Polkadot Solidity Hackathon** is the first Solidity hackathon on Polkadot: a 6-week online program designed to celebrate Solidity coming to Polkadot and accelerate the next generation of smart contract builders.

From **February 15 to March 24, 2026**, developers across APAC will turn ideas into production-ready dApps on **Polkadot Hub**, shipping with:
- EVM Smart Contracts
- PVM Smart Contracts
- Shared security
- Seamless cross-chain capabilities

## About Polkadot Hub
**Polkadot Hub** is a dedicated, unified chain that enables permissionless smart contract deployment and is EVM-compatible, allowing developers to deploy Solidity smart contracts directly on Polkadot.

It consolidates key system functions into one environment:
- DOT token
- Staking
- Governance
- Bridges

Your application on Polkadot Hub can:
- Communicate seamlessly with existing parachains
- Bridge and interact with other EVM-compatible chains
- Leverage XCM (Cross-Consensus Messaging) as a native feature
- Build once and scale across the Polkadot ecosystem and beyond

### Polkadot
Polkadot is a next-generation Layer-0 blockchain protocol designed for a multi-chain future, enabling secure communication and interoperability across blockchains.

Core innovations include:
- Application-specific Layer-1 chains (parachains)
- Shared security via Nominated Proof of Stake (NPoS)
- Cross-chain interactions via XCM
